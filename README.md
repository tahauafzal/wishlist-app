# 🎁 Wishlist App

A web-based wishlist management application where users can create wishlists, share them with friends, and reserve items.

## Features

- ✅ Create digital wishlists
- ✅ Add items with descriptions and links
- ✅ Share wishlists via unique links
- ✅ Reserve items on wishlists

## Tech Stack

- Java 17, Spring Boot 3.1.5
- H2 Database
- Thymeleaf, HTML/CSS
- Maven

## Quick Start

```bash
git clone https://github.com/tahauafzal/wishlist-app.git
cd wishlist-app
mvn clean spring-boot:run
```
